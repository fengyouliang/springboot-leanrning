## 本书的“写法”（不强制模板，但追求可顺读）

- **叙事优先**：把“机制/源码/实验/排障”写成叙事里的插入段，而不是并列条目清单。
- **证据链优先**：重要结论尽量落到一个可跑的 Lab/Test，或至少给出可复现的断点/日志观察点。
- **入口稳定**：书的目录先稳定下来；模块 docs 后续允许渐进式迁移与重排（必要时少量 redirect）。
